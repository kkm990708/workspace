HTML 문서에서 연속된 공백(띄어쓰기, 엔터) 사용시 문제점

HTML에서 연이은 공백은 띄어쓰기 한칸으로 인식됨

공백을 표시하는 방법

1. & nbsp ; : 한 칸 띄어쓰기를 나타내는 문자열(space bar와 다름)
2. br : line break(다음 줄 이동)
3. css 를 이용하는 방법
4. pre 태그를 이용하는 방법

p, pre태그

p(Paragraph : 단란, 문단) : 글을 작성하는 용도, 종료 시 줄 바꿈

1문단 입니다

2문단 입니다

3문단 입니다

pre(Preformatted : 사전 형식 지정) : 태그 내에 작성된 문자열 형식을 그대로 보여주는 태그
(p태그 + 형식 유지) 

      1번          ㅇ빕        닌               다

글자 관련 태그

b 태그 : 단순히 글자를 굵게 표시하는 태그
i 태그 : 단순히 글자를 기울여 표시하는 태그
웹 접근성을 위해 사용되는 태그
      일반인 뿐만 아니라 장애인, 고령자들을 위해
      웹 사이트에서 제공하는 서비스를 모두 이용할 수 있도록 보장하는 것
      (의무사항)
      ex) 스크린 리더
strong 태그 : 글자(단어) 강조하는 태그
em 태그 : 글자(문장) 강조하는 태그
mark 태그 : 형광펜 효과를 나타내는 코드
u 태그 : 밑줄
s 태그 : 취소선(잘못됨)
del 태그 : 삭제선 (이제 없음)
sup 태그(윗첨자) : x2
sub 태그(아랫첨자) : log10
abbr(abbreviation : 약어) 태그 : 마우스 오버 시 툴팁 출력 태그
